jive.ui.Textinput:getValue()

Returns the text displayed in the label.

jive.ui.Textinput:setValue(value)

Sets the text displayed in the label.

jive.ui.Textinput:init(style, value, closure)

Creates a new Textinput widget with initial value value. The <closure> is a function that will be called at the end of the text input. This function should return false if the text is invalid (the window will then bump right) or return true when the text is valid.
