Updates the player with fresh data from SS.

jive.slim.Player:free()

Deletes the player.

jive.slim.Player:getHomeMenuItem()

Returns the home menu menuItem that represents this player. This is used by jive.applet.SlimDiscovery to remove the player from the menu if/when it disappears.

jive.slim.Player:setHomeMenuItem(homeMenuItem)

Stores the main home menuItem that represents this player. This is used by jive.applet.SlimDiscovery to manage the home menu item.

tostring(aPlayer)
