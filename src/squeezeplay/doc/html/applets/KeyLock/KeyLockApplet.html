DESCRIPTION

This applet implements using a key combination to lock the Jive screen.

FUNCTIONS

Applet related methods are described in jive.Applet. KeyLockApplet overrides the following methods:

applets.KeyLock.KeyLockApplet:free()

Overridden to return always false, this ensure the applet is permanently loaded.
