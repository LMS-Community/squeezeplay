DESCRIPTION

An abstract socket that sends/receives data using a NetworkThread. It proposes services to close/free sockets and interface with the main thread, along with convenient proxy functions.

SYNOPSIS

 -- jive.net.Socket is abstract so this is not a useful example
 local mySocket = Socket(jnt, "mySocket")
 -- FROM THE MAIN THREAD
 -- transform a sink into a network thread one
 local t_Sink = mySocket:safeSink(sink)
 -- information about the socket
 log:debug("Freeing: ", tostring(mySocket))
 -- print
 Freeing: Socket {mySocket}
 -- free the socket
 mySocket:free()

FUNCTIONS

jive.net.Socket(jnt, name)

Creates a socket named name to interface with the given jnt (a jive.net.NetworkThread instance). name is used for debugging and defaults to ``''. Must be called by subclasses.

jive.net.Socket:free()

Frees and closes the socket including any reference in the jnt. Must be called by subclasses.

jive.net.Socket:getSafeSinkGenerator()

Returns a function that transforms a main thread sink in a network thread sink. The function accepts an optional boolean indicating if nil must be sent (i.e. standard source behaviour)

jive.net.Socket:safeSink()

Transforms a main thread sink in a network thread sink.

tostring(aSocket)

if aSocket is a jive.net.Socket, prints Socket {name}

jive.net.Socket:perform(func)

A proxy for the perform method of jive.net.NetworkThread.
